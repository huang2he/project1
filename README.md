# csv-to-echarts

## 主要功能：
 ### - shell读取底层数据并导出为带日期的csv文件（数据来源于asterisk）
 ### - jQuery+ajax转csv为数组
 ### - 数组导入echart，生成可视化图表
 ### - 前端采用可响应式布局+侧边栏 支持多个echart的拓展

## 核心库：

下载地址 （https://github.com/apache/echarts）
